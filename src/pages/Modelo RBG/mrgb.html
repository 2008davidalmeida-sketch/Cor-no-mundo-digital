Modelo RGB

O modelo RGB é um modelo de cor aditivo baseado em luz. RGB significa Red (Vermelho), Green (Verde) e Blue (Azul). Todas as cores são criadas combinando diferentes intensidades destas três luzes. Cada cor é representada por valores numéricos (normalmente de 0 a 255)
